# 転送計画の入口

このディレクトリは、ZeroRoomLab-manifest から他リポジトリへ転送する内容の整理場所です。

## 方針

- 反映対象の内容は、まずこのディレクトリ配下で整理する
- ここに置いた内容は「転送候補」扱いとし、対象リポジトリへ送る前に要件を確認する
- すでに他リポジトリに反映済みのものは、こちらのステータスを更新する
- 直接コピーではなく、対象リポジトリの文脈に合わせた正規化を行う

## 運用ルール

1. 生のノートは [note](../) 側に残す
2. 転送候補は [note/transfer_plan](.) 配下で管理する
3. 反映先ごとに、対象ファイル・要点・優先度を記録する
4. 転送後も、元の内容がこのリポジトリに残るようにする
5. 転送結果、検証、未転送範囲を`foldlog/`へreceiptとして残す

詳細は[Noteから公開先へのデータ配信契約](note-delivery-contract.ja.md)を参照する。

## 現在の扱い

- ここに置く内容は、他リポジトリへ反映すべき「候補資料」扱いとする
- いったんこの note 側に置いて、別リポジトリへ移す前の staging として運用する
