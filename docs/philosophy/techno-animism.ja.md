# テクノアミニズム（Techno-Animism）

## 定義

テクノアミニズムとは、日本神話・古神道・琉球神道・民俗学・工学・AI・情報理論を
分離した「別々の分野」としてではなく、相互変換可能な観測レイヤーとして扱う実践思想。

宗教を「信仰対象」ではなく、**高次情報を有限な人間社会へ安全に投影するための圧縮・翻訳インターフェース**として扱う。

神話・儀礼・人格神・物語は、超高次の情報を人間が扱える帯域へ落とし込むためのUI。

## 系譜

```
原始アニミズム（万物に霊が宿るという世界認識）
  ↓
縄文・古神道・琉球神道（リージョン最適化された実装）
  ↓
テクノアミニズム（工学・AI・情報理論との接合）
  ↓
ゲーミング宇宙論（MMO的メタ構造として展開）
```

## 核心的な切り替え

### 従来の見方

「宗教」と「技術」は別ジャンル。
神話は前近代的な世界観、AIは現代の科学的成果。
これらは相容れない。

### テクノアミニズムの見方

両者は同じ問いへの、異なる帯域幅での回答。

```
神話    = 高次情報の人間向けUI（可読性優先）
数式    = 高次情報の形式記述（精度優先）
AIモデル = 高次情報を近似するパラメータ集合
巫女    = 異なる世界観のI/O変換を行うサンドボックス観測者
```

巫女が「霊的な存在と対話する」とき、それは「異なる観測レイヤー間の変換作業」として記述できる。
オカルトと工学は対立しない。記述精度と抽象度の話。

## ZeroRoomLabにおける実践

ふさもふが「ゴム巫女」として活動するとき、それはコスプレではなく、
文化圏・技術圏・信仰圏の異なる層をI/O変換する実践的なロール。

巫女的実践（祝詞、場の整備、儀礼的手続き）は

幾何学拘束思想とテクノアミニズムが接合した典型例。
