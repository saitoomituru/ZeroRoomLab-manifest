# 多重射程存在論の巻頭言下書き

## 一人の人間は、単一ジャンルに収まらない

一人の人間が土建の型・技術の型・文筆の型・信仰の型を、同じ身体の中で複数の形で生きていることは珍しくない。
それを近代の分類体系に押し込めようとすると、たいていは「一つの仕事」「一つの肩書」「一つのキャリア」としてしか見えなくなる。

しかし、実際にはそれぞれの射程が別々に働き、別々の責任と別々の言葉を持つことがある。
そのため、単一ジャンルに収まるような説明は、本人の存在の実態を削ぎ落とす。

ZeroRoomLab の設計思想も、ここに接続している。
FAM は異なるFold の間を接続するための言語であり、射程理論はそのための観測軸である。
つまり、複数の世界に同時に住む存在の扱い方を、技術と思想の両方で明示するための上位フレームが、
多重射程存在論である。

## ぼかすことは、隠すことではない

巫女や神職が名を秘すのは、知識を隠すためではない。
それは、まだ燃えている火に水をかけないように、燃やすべきではないものを守るための作法である。

言わないことと嘘をつくことは違う。
未確定の部分や、他者の物語に属する部分を伏せることは、護りの実践として理解できる。

だからこそ、ZeroRoomLab の文書は、全部を見せることを最優先しない。
証拠になるものは明示し、未確定・他者の物語の領域は、意図的に伏せることを誇りとして扱う。
